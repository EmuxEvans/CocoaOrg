* Supported Syntax (so far) [9/9]
** DONE Keywords (aka In Buffer Settings)
   CLOSED: [2016-09-03 Sat 12:47]
   
   #+BEGIN_SRC org
   #+TITLE: Hello World
   #+OPTIONS: Hello World
   #+END_SRC

** DONE Headlines
   CLOSED: [2016-09-03 Sat 12:47]
   #+BEGIN_SRC org
   * Head Line 1
   * Head Line 2
   ** Head Line 2.1
   *** Head Line 2.1.1
   #+END_SRC

** DONE Paragraph
   CLOSED: [2016-09-03 Sat 12:47]
   Lines without *line breaker* becomes a *paragraph*.

** DONE Emphasis
   CLOSED: [2016-09-03 Sat 12:47]
   #+BEGIN_SRC org
   *bold* 
   /italic/
   _underlined_
   =verbatim=
   ~code~
   +strike-through+
   #+END_SRC

** DONE Link
   CLOSED: [2016-09-03 Sat 12:47]
   #+BEGIN_SRC org
   [[google][https://www.google.com]]
   #+END_SRC

** DONE List
   CLOSED: [2016-09-03 Sat 12:47]
   #+BEGIN_SRC org
   # ordered list
   1. first
   2) second
   3. 3rd
   
   # unordered list
   - item
   + item
   * item

   # nested list
   - item
     1. sub item
     1) sub item
   - item
   #+END_SRC

** DONE Horizontal rules
   CLOSED: [2016-09-03 Sat 12:47]
   #+BEGIN_SRC org
   Above.
   -----
   Below
   #+END_SRC

** DONE Comment
   CLOSED: [2016-09-03 Sat 12:47]
   #+BEGIN_SRC org
   # This is a comment.
   #This is a regular line.
   #+END_SRC

** DONE Blocks
   CLOSED: [2016-09-03 Sat 12:47]
   #+BEGIN_SRC javascript
     Console.log("Hello Org.")
   #+END_SRC
